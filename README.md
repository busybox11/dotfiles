## How to install
### Required packages
- i3-gaps
- compton-tryone
- pywal
- feh
- python3 + python3-pip
- spotipy (pip3)
- pulseaudio

Check [busybox11/spotify-polybar](https://github.com/busybox11/spotify-polybar) to install only my polybar config with my Spotify module.
